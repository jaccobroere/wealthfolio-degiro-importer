# DEGIRO Importer 1.1.0

## Highlights

- Updated for the Wealthfolio 3.6.1 addon contract.
- Added explicit instrument review with account-scoped mappings.
- Added duplicate and overlap protection before import writes.
- Improved DEGIRO localized quantity, amount, tax, fee, and accrued-interest
  handling.
- Added deterministic addon packages, checksums, and release validation.

## Compatibility

- Wealthfolio 3.6.1
- DEGIRO Account statement CSV exports in Dutch or English header variants

See the [installation guide](../INSTALL.md) and [input format](../FORMAT.md)
before importing.
