# 1.1.0 release record

## Status

**BLOCKED — not published.** This is a preparation record, not a release
announcement, tag, or artifact.

## Blocking evidence

T09 has not yet proved the accrued-interest representation on a supported host
instrument, the associated host import/write and cash/cost-basis outcomes, or
the `saveMany` partial-versus-atomic behavior. Those proof items must close
before a release can be approved.

## Future artifact convention

The approved release artifact will be named
`wealthfolio-degiro-importer-1.1.0.zip`. Its checksum will be recorded in the
same release's `SHA256SUMS`; consumers must verify the ZIP before installation.
Once published, the version tag and artifacts are immutable.
