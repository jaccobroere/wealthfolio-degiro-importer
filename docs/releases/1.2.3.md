# DEGIRO Importer 1.2.3

## Highlights

- **Faster mapping review.** Confirm all unambiguous security mappings in one
  action while retaining individual review where it is needed.
- **Complete accrued-interest imports.** The related cash settlement is now
  represented as a fee activity instead of blocking the import.
- **More reliable imports.** Canonical activities returned by Wealthfolio's
  import check are preserved through saving and mappings remain account scoped.
- **Simpler releases.** Release publication validates the public fixture suite,
  package contents, and deterministic artifacts without a self-attested local
  result.

## Compatibility

- Wealthfolio: 3.6.1
- Add-on SDK: 3.6.1
- DEGIRO account-statement CSV exports in Dutch or English header variants
