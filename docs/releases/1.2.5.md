# DEGIRO Importer 1.2.5

## Highlights

- **Supported import workflow.** Confirmed, host-reviewed activities now use
  Wealthfolio's `activities.import` API instead of the generic bulk editor
  endpoint.
- **Reliable duplicate handling.** New imports use Wealthfolio's native
  import duplicate detection, while pre-existing importer metadata remains
  protected during the transition.
- **Safer final outcomes.** The importer preserves reviewed asset mappings,
  posts only after confirmation, and presents incomplete host results without
  automatically retrying a potentially ambiguous batch.

## Compatibility

- Wealthfolio: 3.6.1
- Add-on SDK: 3.6.1
- DEGIRO account-statement CSV exports in Dutch or English header variants
