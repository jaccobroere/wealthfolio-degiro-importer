# DEGIRO Importer 1.2.2

## Highlights

- **Restored Wealthfolio 3.6.1 navigation.** The importer now registers its
  sidebar entry at runtime and uses the host's `/addon/degiro-importer` route.
  The prior package used a declarative navigation model that this host version
  does not support, leaving the importer inaccessible after installation.

## Compatibility

- Wealthfolio: 3.6.1
- Add-on SDK: 3.6.1
- DEGIRO Account statement CSV exports in Dutch or English header variants

## Notes

- No change to CSV parsing, mapping, reconciliation, or import semantics.
- Reinstall this ZIP from **Settings → Add-ons** to replace an earlier package.
