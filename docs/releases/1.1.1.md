# DEGIRO Importer 1.1.1

## Highlights

- Modernized CI/CD pipelines to Node 22 and the latest GitHub Actions.
- Fixed release workflow: pnpm is now installed before the setup-node cache
  lookup, resolving the "Unable to locate executable file: pnpm" failure.
- Fixed the release attestation: `releaseCommit` and `zipSha256` are now
  computed at runtime by the workflow and published in a final
  `attestation.json` alongside the ZIP, instead of carrying unsolvable
  placeholders in the committed schema.

## Compatibility

- Wealthfolio 3.6.1
- DEGIRO Account statement CSV exports in Dutch or English header variants

See the [installation guide](../INSTALL.md) and [input format](../FORMAT.md)
before importing.
