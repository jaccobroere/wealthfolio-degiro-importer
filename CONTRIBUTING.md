# Contributing

Thanks for helping improve the DEGIRO importer.

Start with the [development guide](docs/DEVELOPMENT.md). It covers the supported
toolchain, public synthetic verification, local-only acceptance, fixtures, and
the disposable Wealthfolio host.

Please keep pull requests focused and include tests for parser or import
behavior changes. Use synthetic data in commits, issues, and pull requests;
never attach a real broker statement or a redacted derivative of one.

User-facing behavior changes should update the README or the relevant document
under `docs/`. Release changes also require the maintainer release checklist.
